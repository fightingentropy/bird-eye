Command configuration

Manage the bird command that powers the feed.

Command config

Configure the bird command used for the feed.

Allowed: bird list-timeline, bird home, --following, -n

Command: --

Allowed commands

Command

  • bird list-timeline <list> — list timeline (default)
  • bird home — home timeline (For You)
  • bird home --following — following feed

Flags

  • -n <count> or --count <count> — 1 to 100
  • --json — optional; ignored if included

Examples

bird list-timeline 1933193197817135501 -n 100 --json
bird home --following -n 20
bird home -n 5
